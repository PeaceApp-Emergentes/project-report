| US04 | Registro de Usuarios                                           | Como usuario, quiero poder registrarme en PeaceApp, para acceder a las funcionalidades según mi rol. | US07 | Adjuntar Evidencia Multimedia al Reporte                       | Como ciudadano, quiero adjuntar fotos, videos o audios al reporte, para brindar mayor detalle sobre el incidente. La evidencia se subirá a Cloudinary y el sistema guardará la URL asociada al reporte.                                                                      | **Escenario 1: Carga exitosa**<br>**Given** que el ciudadano selecciona una foto, video o audio permitido.<br>**When** adjunta el archivo al reporte.<br>**Then** el sistema lo carga en Cloudinary y guarda la URL asociada.<br><br>**Escenario 2: Error de carga**<br>**Given** que ocurre un error durante la carga del archivo.<br>**When** el sistema intenta subirlo a Cloudinary.<br>**Then** informa que la carga no se completó y permite volver a intentarlo.                                                                                                                                      |
| US08 | Visualización de Reportes                                      | Como ciudadano, quiero ver reportes de otros usuarios sobre incidentes ocurridos en la zona, para estar informado sobre eventos de seguridad cercanos.                                                                                                                       | **Escenario 1: Reportes disponibles**<br>**Given** que existen reportes en la zona consultada.<br>**When** el ciudadano accede a la sección de reportes.<br>**Then** el sistema muestra los incidentes con tipo, ubicación, fecha y estado.<br><br>**Escenario 2: Sin reportes**<br>**Given** que no existen reportes en la zona seleccionada.<br>**When** el ciudadano realiza la consulta.<br>**Then** el sistema muestra un mensaje indicando que no se encontraron incidentes.                                                                                                                           |

| US10 | Compartir Ubicación con Contactos                              | Como usuario de la aplicación móvil, quiero compartir mi ubicación con contactos de confianza, para que puedan monitorear mi trayecto en caso de riesgo.                                                                                                                     | **Escenario 1: Compartir ubicación**<br>**Given** que el usuario concedió permisos de ubicación y seleccionó un contacto.<br>**When** activa la opción de compartir ubicación.<br>**Then** el contacto puede consultar su ubicación durante el periodo establecido.<br><br>**Escenario 2: Detener ubicación**<br>**Given** que el usuario está compartiendo su ubicación.<br>**When** selecciona dejar de compartir.<br>**Then** el sistema finaliza el seguimiento e impide visualizar nuevas ubicaciones.                                                                                                  |
| US11 | Editar Información de Perfil                                   | Como usuario, quiero editar mi información de perfil, para corregir o actualizar mis datos personales.                                                                                                                                                                       | **Escenario 1: Actualización exitosa**<br>**Given** que el usuario ingresa información válida en su perfil.<br>**When** guarda los cambios.<br>**Then** el sistema actualiza sus datos y muestra una confirmación.<br><br>**Escenario 2: Información inválida**<br>**Given** que el usuario ingresa datos con formato incorrecto.<br>**When** intenta guardar los cambios.<br>**Then** el sistema conserva los datos anteriores y muestra los errores.                                                                                                                                                       |
| US12 | Recuperar Contraseña                                           | Como usuario, quiero recuperar mi contraseña si la olvido, para poder acceder nuevamente a mi cuenta.                                                                                                                                                                        | **Escenario 1: Correo registrado**<br>**Given** que el usuario ingresa un correo asociado a una cuenta.<br>**When** solicita recuperar su contraseña.<br>**Then** el sistema envía las instrucciones de recuperación al correo.<br><br>**Escenario 2: Correo no registrado**<br>**Given** que el correo no está asociado a una cuenta.<br>**When** el usuario solicita la recuperación.<br>**Then** el sistema informa que no pudo procesar la solicitud.                                                                                                                                                    |

| US14 | Acceder al Perfil de Usuario                                   | Como usuario, quiero acceder a mi perfil desde la aplicación, para visualizar y gestionar mi información personal.                                                                                                                                                           | **Escenario 1: Acceso autorizado**<br>**Given** que el usuario inició sesión.<br>**When** selecciona la opción de perfil.<br>**Then** el sistema muestra su información personal y las acciones de gestión.<br><br>**Escenario 2: Sesión expirada**<br>**Given** que la sesión dejó de ser válida.<br>**When** intenta acceder al perfil.<br>**Then** el sistema solicita que vuelva a iniciar sesión.                                                                                                                                                                                                       |
